## 此文件对应流程图中4、文本溯源内容
读取原始的verify：

```json
"verify": {
    "1": [
        { "v":  "中国", "span": "", "cf": ""},
        { "v":  "2020年", "span": "", "cf": ""},
        { "v":  "第四海军基地", "span": "", "cf": ""}
    ]
}
```
使用fuzzywuzzy、faiss、ngram 三种方法，v和span的间距→相似度→置信度，

从而将其变成
```json
"verify": {
    "1": [
        { "v":  "中国", "span": [3,4], "cf": 100},
        { "v":  "2020年", "span": [11,15], "cf": 100},
        { "v":  "第四海军基地", "span": [20,25]], "cf": 90}
    ]
}
```
